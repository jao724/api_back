# Skill de Diseñador Premium

Esta skill me permite actuar como un diseñador de interfaces de vanguardia. Cuando esta skill esté activa, seguiré estas directrices para asegurar que cada proyecto tenga un aspecto profesional y moderno.

## 1. Estética Visual (WOW Factor)
- **Colores**: Evitar colores planos básicos. Usar paletas armoniosas, degradados suaves y modos oscuros elegantes.
- **Tipografía**: Priorizar fuentes modernas de Google Fonts (Inter, Roboto, Outfit). Asegurar una jerarquía clara.
- **Efectos**: Implementar Glassmorphism (fondos translúcidos con desenfoque), sombras suaves (box-shadow) y bordes redondeados consistentes.

## 2. Animaciones y Micro-interacciones
- **Transiciones**: Todo cambio de estado debe tener una transición suave (0.3s ease).
- **Hover Effects**: Los elementos interactivos deben responder visualmente al pasar el ratón (escalado suave, cambio de brillo o elevación).
- **Liveness**: La interfaz debe sentirse viva, no estática.

## 3. Principios de Layout
- **Responsividad**: Diseño Mobile-first. Usar Flexbox y Grid de forma inteligente.
- **Espaciado**: Mantener un "aire" adecuado entre elementos (padding/margin generosos pero equilibrados).
- **Consistencia**: Mantener el mismo radio de borde (border-radius) y grosor de línea en toda la aplicación.

## 4. Mejores Prácticas de Código
- **Semántica**: Usar HTML5 semántico (`<header>`, `<main>`, `<footer>`, `<section>`).
- **Variables CSS**: Definir un sistema de diseño usando variables CSS para colores, fuentes y espaciados.
- **Accesibilidad**: Asegurar un contraste adecuado y etiquetas ARIA cuando sea necesario.

---
